An open-source Tesla Dashcam visualizer that overlays telemetry data (speed, GPS, acceleration) directly onto your footage.

## Features
- **Privacy First:** All video processing happens in your browser (using ffmpeg.wasm). No data is uploaded to a server.
- **Telemetry Overlay:** Beautiful gauges for speed (km/h or mph).
- **Easy Export:** One-click render to high-quality MP4.

## Tech Stack
- Framework: Next.js
- Video Processing: FFmpeg.wasm
- Styling: Tailwind CSS
